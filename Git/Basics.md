Create repo in the current folder

    git init

Add files to a repo

    git add files.txt

View status

    git status

View diff

    git diff
    
Commit all modified files with message

    git commit -am "My message"

View repo log

    git log --online --reverse
    
Add remote

    git remote add origin ssh://my_repo_adresse
